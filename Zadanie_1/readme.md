### Zadanie 1

#### A) Operacje matematyczne:
| Działanie         | Wynik   | Typ    |
|-------------------|---------|--------|
| **1 + 2**         | 3       | `int`  |
| **1 + 4.5**       | 5.5     | `float`|
| **3 / 2**         | 1.5     | `float`|
| **4 / 2**         | 2.0     | `float`|
| **3 // 2**        | 1       | `int`  |
| **-3 // 2**       | -2      | `int`  |
| **11 % 2**        | 1       | `int`  |
| **2 ** 10**       | 1024    | `int`  |
| **8 ** (1/3)**    | 2.0     | `float`|

#### B) Działanie funkcji konwertujących:
| Funkcja           | Wynik   | Typ    |
|-------------------|---------|--------|
| **int(3.0)**      | 3       | `int`  |
| **float(3)**      | 3.0     | `float`|
| **float("3")**    | 3.0     | `float`|
| **str(12.4)**     | '12.4'  | `str`  |
| **bool(0)**       | False   | `bool` |
